# CLAP_Plugin

Плагин-ассистент для обработки, улучшения звука

## Функционал, фичи
- Анализ входного сигнала
- Различные настройки и встроенные конфигурации для разных типов звука
- Автоматическое определение типа звука (возможно с использованием машинного обучения)
- Визуализация звука, а также разницы исходного и улучшенного звука
- Возможно дополнительные визуальные фишки, подсказки и т.п.

## Стек
- C++
- CLAP(CLever Audio Plug-in)
